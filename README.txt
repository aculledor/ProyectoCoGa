Proyecto realizado por Abraham Trashorras Rivas, estudiante de GrEI en la USC.

Para el manejo del sonido se usa la libreria irrKlang.h cuya autoría corresponde a https://www.ambiera.com/irrklang/downloads.html
y cuyo es explicado en https://learnopengl.com/In-Practice/2D-Game/Audio.

Ninguna otra parte del código es importada, el resto esta desarrollado por mi durante practicas anteriores
y el desarrollo de este proyecto además de los códigos de ejemplos proporcionados por la materia de
Computación Gráfica.


################
## CONTROLES: ##
###############################################################################################
Moverse-> 	w, s avanzar y retroceder
		a, d girar el Avatar

Reiniciar posición-> r

Parar cualquier sonido que se este reproduciendo-> m

Activar la banda sonora-> n
